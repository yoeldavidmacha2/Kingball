# 🎱 Kingball - Listero App

Una aplicación web moderna para gestionar jugadas de lotería con interfaz optimizada para móviles.

## ✨ Características

- 🔐 **Autenticación**: Sistema de login seguro
- 📱 **Responsive Design**: Optimizado para cualquier dispositivo
- 🎰 **Sistema de Jugadas**: Bolas, Parles y Centenas
- 💰 **Cálculo Automático**: Totales en tiempo real
- 🌓 **Modo Día/Noche**: Cambio automático según la hora
- ⌨️ **Teclado Numérico**: Entrada rápida de datos
- 📋 **Menú Lateral**: Navegación intuitiva

## 🚀 Inicio Rápido

### Opción 1: Abrir directamente en el navegador
```
Simplemente abre el archivo index.html en tu navegador
```

### Opción 2: En Netlify (Recomendado)
1. Ve a https://netlify.com
2. Conecta este repositorio de GitHub
3. Netlify deployará automáticamente
4. Tu app estará disponible en `https://kingball-app.netlify.app`

## 📖 Cómo usar

1. **Login**: Ingresa un usuario y contraseña
2. **Ingresa números**: Usa el teclado numérico
3. **Separa con guión**: Después del número, presiona `-` para ingresar el monto
4. **Envía**: Presiona "Enviar" para registrar la jugada
5. **Visualiza**: Los totales se calculan automáticamente

## 🎯 Tipos de Jugadas

- **Bola**: Número de 2 dígitos con monto fijo y corrido
- **Parle**: Múltiples números con un monto compartido
- **Centena**: Número de 3 dígitos
